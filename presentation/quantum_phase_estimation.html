01

Medir la Fase con Qubits de Apoyo

El origen físico de la Estimación de Fase Cuántica

El Problema de la Fase Global

Dado un operador unitario $U$ y un eigenestado $|u\rangle$ tal que $U |u\rangle = e^{2\pi i \theta} |u\rangle$, queremos conocer la fase $\theta$.

La fase $e^{2\pi i \theta}$ actúa como una fase global en el estado del sistema. Si medimos el sistema directamente, la fase desaparece: $|\langle u | u \rangle|^2 = 1$. Es físicamente indetectable.

La Solución: El Qubit de Apoyo (Referencia)

Necesitamos un sistema de referencia externo. Introducimos un **qubit de apoyo** (auxiliar) preparado en superposición $|+\rangle = \frac{|0\rangle+|1\rangle}{\sqrt{2}}$ y aplicamos una operación controlada-$U$:

$$|0\rangle |u\rangle \xrightarrow{H \otimes I} \frac{|0\rangle + |1\rangle}{\sqrt{2}} \otimes |u\rangle \xrightarrow{c\text{-}U} \frac{|0\rangle |u\rangle + |1\rangle U|u\rangle}{\sqrt{2}} = \left(\frac{|0\rangle + e^{2\pi i \theta}|1\rangle}{\sqrt{2}}\right) \otimes |u\rangle$$

Midiendo con 1 Qubit de Apoyo

Si la fase tiene resolución de 1 bit, se escribe en binario fraccionario como $\theta = 0.\theta_1 = \frac{\theta_1}{2}$ donde $\theta_1 \in \{0, 1\}$.

$$|\Psi\rangle_{\text{apoyo}} = \frac{|0\rangle + e^{2\pi i (0.\theta_1)}|1\rangle}{\sqrt{2}} = \frac{|0\rangle + e^{i\pi \theta_1}|1\rangle}{\sqrt{2}}$$
  • Si $\theta_1 = 0 \implies \theta = 0 \implies |\Psi\rangle_{\text{apoyo}} = |+\rangle$. Aplicar Hadamard ($H$) lo lleva a $|0\rangle$.
  • Si $\theta_1 = 1 \implies \theta = 0.5 \implies |\Psi\rangle_{\text{apoyo}} = |-\rangle$. Aplicar Hadamard ($H$) lo lleva a $|1\rangle$.

En 1 qubit, aplicar **Hadamard** equivale a la Transformada Inversa para decodificar la fase.

Midiendo con 2 Qubits de Apoyo

Para mayor precisión, supongamos $\theta = 0.\theta_1 \theta_2 = \frac{\theta_1}{2} + \frac{\theta_2}{4}$. Necesitamos 2 qubits de apoyo ($c_0$ y $c_1$) en $|+\rangle$:

1

Qubit $c_1$ (peso $2^{-2}$): Aplicamos Controlled-$U^{2^1} = c\text{-}U^2$. La fase aplicada es $2\theta = 2(0.\theta_1 \theta_2) = \theta_1.\theta_2 \equiv 0.\theta_2 \pmod 1$. El estado queda: $\frac{|0\rangle + e^{2\pi i 0.\theta_2}|1\rangle}{\sqrt{2}}$.

2

Qubit $c_0$ (peso $2^{-1}$): Aplicamos Controlled-$U^{2^0} = c\text{-}U^1$. La fase aplicada es $1\theta = 0.\theta_1 \theta_2$. El estado queda: $\frac{|0\rangle + e^{2\pi i 0.\theta_1 \theta_2}|1\rangle}{\sqrt{2}}$.

El estado conjunto de los qubits de apoyo es:

$$|\Psi\rangle_{\text{apoyos}} = \frac{|0\rangle + e^{2\pi i 0.\theta_2}|1\rangle}{\sqrt{2}} \otimes \frac{|0\rangle + e^{2\pi i 0.\theta_1 \theta_2}|1\rangle}{\sqrt{2}}$$

Midiendo con 3 Qubits de Apoyo

Para una fase de 3 bits $\theta = 0.\theta_1 \theta_2 \theta_3$, usamos 3 qubits de apoyo y aplicamos potencias en cascada de $U$ ($U^4, U^2, U^1$):

  • Qubit $c_2$ (peso $2^{-3}$): Aplicamos $c\text{-}U^4 \to 4\theta = 2\theta_1 + \theta_2 + \frac{\theta_3}{2} \equiv 0.\theta_3 \pmod 1$. Estado: $\frac{|0\rangle + e^{2\pi i 0.\theta_3}|1\rangle}{\sqrt{2}}$.
  • Qubit $c_1$ (peso $2^{-2}$): Aplicamos $c\text{-}U^2 \to 2\theta \equiv 0.\theta_2 \theta_3 \pmod 1$. Estado: $\frac{|0\rangle + e^{2\pi i 0.\theta_2 \theta_3}|1\rangle}{\sqrt{2}}$.
  • Qubit $c_0$ (peso $2^{-1}$): Aplicamos $c\text{-}U^1 \to 1\theta = 0.\theta_1 \theta_2 \theta_3$. Estado: $\frac{|0\rangle + e^{2\pi i 0.\theta_1 \theta_2 \theta_3}|1\rangle}{\sqrt{2}}$.

El estado conjunto del registro de apoyos es:

$$|\Psi\rangle_{\text{apoyos}} = \frac{|0\rangle + e^{2\pi i 0.\theta_3}|1\rangle}{\sqrt{2}} \otimes \frac{|0\rangle + e^{2\pi i 0.\theta_2 \theta_3}|1\rangle}{\sqrt{2}} \otimes \frac{|0\rangle + e^{2\pi i 0.\theta_1 \theta_2 \theta_3}|1\rangle}{\sqrt{2}}$$

La Equivalencia con la QFT

Recordemos la definición matemática de la **Transformada de Fourier Cuántica (QFT)** para un estado base binario $|x_1 x_2 \dots x_n\rangle$:

$$QFT |x_1 x_2 \dots x_n\rangle = \frac{|0\rangle + e^{2\pi i 0.x_n}|1\rangle}{\sqrt{2}} \otimes \dots \otimes \frac{|0\rangle + e^{2\pi i 0.x_1 x_2 \dots x_n}|1\rangle}{\sqrt{2}}$$

La Revelación Clave:

El estado que generamos en el registro de apoyos al aplicar compuertas controladas en cascada ($U^{2^j}$) es **exactamente** la QFT de la fase $|\theta_1 \theta_2 \dots \theta_n\rangle$.

Por lo tanto, para decodificar las fases relativas y obtener el valor binario legible en el laboratorio, debemos aplicar obligatoriamente la Transformada de Fourier Cuántica Inversa ($QFT^\dagger$).

Circuitos e Implementaciones

Construcción paso a paso de los circuitos en Qiskit

1 Qubit: Circuito y Código en Qiskit

from qiskit import QuantumCircuit
import numpy as np

# 1 qubit de control (0) + 1 qubit objetivo (1)
qc = QuantumCircuit(2, 1)

# Superposición en control
qc.h(0)

# Preparar eigenestado |1> en objetivo
qc.x(1)

# CP (Control-Phase) con theta = 0.5 (fase = pi)
qc.cp(np.pi, 0, 1)

# QFT Inversa de 1 qubit (Hadamard)
qc.h(0)

# Medida
qc.measure(0, 0)

2 Qubits: Circuito y Código en Qiskit

from qiskit import QuantumCircuit
from qiskit.circuit.library import QFTGate
import numpy as np

# 2 qubits control (0, 1) + 1 objetivo (2)
qc = QuantumCircuit(3, 2)

for q in range(2):
    qc.h(q)
qc.x(2) # Estado objetivo |1>

# Cascada controlada (fase theta = 1/4)
fase = 2 * np.pi / 4
qc.cp(2 * fase, 1, 2) # c-U^2 (control 1)
qc.cp(fase, 0, 2)     # c-U   (control 0)

# QFT Inversa de 2 qubits (Decodificación)
qc.append(QFTGate(2).inverse(), [0, 1])

qc.measure([0, 1], [0, 1])

3 Qubits: Circuito y Código de Qiskit

from qiskit import QuantumCircuit
from qiskit.circuit.library import QFTGate
import numpy as np

# 3 qubits control (0, 1, 2) + 1 objetivo (3)
qc = QuantumCircuit(4, 3)

for q in range(3):
    qc.h(q)
qc.x(3) # Objetivo en |1>

# Cascada controlada (fase theta = 1/8)
fase = 2 * np.pi / 8
qc.cp(4 * fase, 2, 3) # c-U^4 (control 2)
qc.cp(2 * fase, 1, 3) # c-U^2 (control 1)
qc.cp(fase, 0, 3)     # c-U   (control 0)

# QFT Inversa de 3 qubits (Decodificación)
qc.append(QFTGate(3).inverse(), [0, 1, 2])

qc.measure([0, 1, 2], [0, 1, 2])

Matemáticas de la Interferencia

Demostración algebraica de por qué colapsa en el valor correcto

El rol de la QFT Inversa en 3 Qubits

Para $N=8$ estados, la QFT Inversa se define formalmente como:

$$QFT^\dagger |k\rangle = \frac{1}{\sqrt{8}} \sum_{j=0}^7 e^{-2\pi i j k / 8} |j\rangle$$

Supongamos que la fase del sistema es exactamente $\theta = 3/8$ (binario $0.011_2$, es decir, estado $|3\rangle$). El estado que entra al decodificador es:

$$|\psi\rangle = \frac{1}{\sqrt{8}} \sum_{k=0}^7 e^{2\pi i (3/8) k} |k\rangle$$

Interferencia Constructiva vs Destructiva

Aplicando el operador de decodificación al estado codificado anterior:

$$QFT^\dagger |\psi\rangle = \frac{1}{8} \sum_{j=0}^7 \left( \sum_{k=0}^7 e^{2\pi i \frac{(3 - j) k}{8}} \right) |j\rangle$$

Evaluamos la suma geométrica interna $\sum_{k=0}^7 r^k$ con $r = e^{2\pi i (3-j)/8}$:

  • Si $j = 3$ (coincidencia): $r = e^0 = 1 \implies \sum_{k=0}^7 1 = 8$. La amplitud del estado $|3\rangle$ es $\frac{8}{8} = 1$ (Interferencia Constructiva).
  • Si $j \neq 3$ (desajuste): $r \neq 1 \implies \frac{1 - r^8}{1 - r} = \frac{1 - e^{2\pi i (3-j)}}{1 - r} = \frac{1 - 1}{1 - r} = 0$ (Interferencia Destructiva total).

El estado colapsa únicamente en $|3\rangle = |011\rangle$ con probabilidad 1.

Generalización y Shor

Generalización matemática para n qubits y Shor

Generalización de QFT para $n$ Qubits

Para un registro de $n$ qubits, cualquier estado computacional $|x\rangle = |x_1 x_2 \dots x_n\rangle$ se mapea a la superposición factorizada de fases:

$$QFT_{2^n} |x\rangle = \bigotimes_{j=1}^n \frac{|0\rangle + e^{2\pi i 0.x_j x_{j+1} \dots x_n} |1\rangle}{\sqrt{2}}$$

Propiedades Clave de la Generalización:

Complejidad de Compuertas: El circuito de QFT requiere $\mathcal{O}(n^2)$ compuertas elementales (Hadamards y rotaciones de fase controladas $R_k$).

Ventaja Cuántica: La transformada de Fourier discreta clásica requiere $\mathcal{O}(n 2^n)$ operaciones aritméticas. Esto representa una aceleración exponencial.

QPE Generalizado y Shor

El algoritmo de estimación de fase cuántica (QPE) utiliza esta estructura general para estimar la fase $\theta$ del operador modular de Shor $U_a |y\rangle = |a y \pmod N\rangle$:

$$U_a |u_s\rangle = e^{2\pi i (s/r)} |u_s\rangle \implies \theta = \frac{s}{r}$$
Flujo de Shor mediante QPE Generalizado:
  1. Preparamos el registro de control de $t$ qubits en superposición uniforme usando Hadamards.
  2. Aplicamos la cascada controlada de operadores modulares: $U^{2^j}_a$ para cada qubit de control $j$.
  3. Aplicamos la $QFT^\dagger$ generalizada para decodificar las fases relativas.
  4. Medimos el registro de control para obtener $x \approx 2^t (s/r)$ y deducir el periodo $r$ clásicamente.
