Instalación y Ejecución Local (`git clone`)

bash — Guía rápida de configuración en local
# 1. Clonar el repositorio
git clone https://github.com/ui4054/qaoa_training.git
cd qaoa_training
# 2. Crear y activar ambiente conda unificado
conda env create -f environment.yml
conda activate qaoa_knapsack
# 3. Lanzar Jupyter Lab en el directorio de notebooks
jupyter lab notebooks/
